# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

## 🎨 UI Theme Description

This project uses a modern Dark Glassmorphism UI theme with Indigo–Purple Neon accents, designed for a premium AI/SaaS dashboard feel.

The interface is based on a black/dark background (#000000) with soft glass-like translucent cards using blurred backgrounds and low-opacity white overlays. UI elements use backdrop-blur effects with subtle borders to create a frosted glass appearance.

The main accent colors are a gradient from Indigo to Purple, used for primary buttons, highlights, logos, and interactive elements. This gives the UI a futuristic and AI-tool inspired look.

To add depth, the design includes soft glowing background orbs (indigo and purple blurred circles) positioned around the screen, creating an ambient neon lighting effect.

Typography is clean and minimal, using white and zinc/gray tones for hierarchy and readability on dark backgrounds.

Buttons and interactive elements include smooth micro-animations such as slight scaling on hover and click, along with soft transitions for a polished user experience.

Overall, the theme represents a:
Dark SaaS + Glassmorphism + Neon Gradient + AI Dashboard aesthetic, similar to modern tools like Notion Dark Mode, Vercel dashboard, and AI platforms.
